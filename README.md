#### 软件架构
前端：vue2+uniapp+javaScript

#### 安装教程

1.  下载本项目
2.  npm install or yarn install 下载前端项目依赖
3.  使用hbuilder运行到小程序模拟器

#### 使用说明

1.  本项目是仿照qq音乐,但是加入了许多自己的想法,主要是对vue2和uniapp的练习。
2.  这个项目仅供学习使用，未经原作者同意不可商用
